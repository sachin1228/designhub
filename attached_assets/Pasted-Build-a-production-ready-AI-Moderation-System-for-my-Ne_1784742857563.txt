Build a production-ready AI Moderation System for my Next.js community platform.

The moderation system must run on the backend before any user-generated content is saved or uploaded.

==================================================
CONTENT TO MODERATE
==================================================

Moderate all user-generated content:

- Chat messages
- Posts
- Comments
- Usernames
- User bios
- Community names
- Image uploads

==================================================
TEXT MODERATION
==================================================

Use OpenAI Moderation API (omni-moderation-latest).

For every text submission:

1. Send the content to OpenAI Moderation.
2. If flagged for unsafe content:
   - Reject immediately.
   - Do not save to the database.
   - Return:
     "This content violates our community guidelines."

If OpenAI passes:

Apply custom business rules.

Reject:

- Referral spam
- Crypto promotions
- Fake giveaways
- Investment scams
- Phishing
- Malware links
- Adult advertisements
- Casino/Gambling spam
- Excessive self-promotion
- Link spam
- Duplicate/repeated messages

Allow:

- Design discussions
- Portfolio links
- Figma links
- Behance
- Dribbble
- GitHub
- Hiring posts
- Job discussions
- Design critiques
- Constructive feedback

==================================================
IMAGE MODERATION
==================================================

Every uploaded image must be scanned BEFORE uploading to Cloudflare R2.

Pipeline:

User Upload

↓

OpenAI Moderation

↓

If passed

↓

NudeNet

↓

If passed

↓

Upload to Cloudflare R2

↓

Save metadata

Reject immediately if:

- Explicit nudity
- Sexual activity
- Pornography
- Child sexual content
- Graphic gore
- Hate symbols

Flag for manual review:

- Artistic nudity
- Medical images
- Educational anatomy
- Breastfeeding
- Statues

Allow:

- UI screenshots
- Design work
- Logos
- Avatars
- Product images
- Portfolio images
- Travel photos

==================================================
NUDENET
==================================================

Create a Python FastAPI microservice.

Endpoint:

POST /moderate-image

Input:

multipart/form-data

Return:

{
  "explicit": false,
  "confidence": 0.98,
  "detections": []
}

==================================================
BACKEND MODERATION SERVICE
==================================================

Create reusable services:

moderateText(text)

Returns:

{
  allowed: boolean,
  reason: string
}

moderateImage(file)

Returns:

{
  allowed: boolean,
  reason: string
}

All API endpoints must use these services before saving content.
